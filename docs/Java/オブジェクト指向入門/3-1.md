:::info 覚える

- **実行用**クラスと**設計図**クラス がある
- **属性**
  - なんのデータを持っているか
- **操作**
  - なんの処理ができるか
- **オブジェクト**
  - クラスから作られる実体(モノ)
- **オブジェクト指向**
  - オブジェクトを中心にしてプログラムを作る考え方

:::

## メリット

- main メソッドの中だとミスが起きやすい
- 開発効率の向上
  - コードを記述を分担できる
- 保守性の向上
  - システムごとに分かれている
  - 機能追加も楽
- メモリ効率向上
  - 必要な処理に、必要な処理分だけのメモリを充てられる
- システム化が容易に
  - 属性と操作という 2 つの情報に分けたことによって、現実を置き換える際のことを考えやすくなった
