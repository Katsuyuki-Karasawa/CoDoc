# 3-9 演習問題

## 1

```java
class Car1 {
    int no;
    int speed;

    void setNo(int n) {
        no = n;
    }
    void run(int s) {
        speed = s;
    }
    void brake() {
        speed = 0;
    }
    void break(int s) {
        speed = s
    }
    void display() {
        System.out.prinln("ナンバー" + no + "の速度は" + speed + "です。")
    }
}
```
